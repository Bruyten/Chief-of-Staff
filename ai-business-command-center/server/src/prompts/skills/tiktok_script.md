# SKILL: tiktok_script

## OBJECTIVE
Write a 25-45 second TikTok script that hooks in 2 seconds, delivers one idea,
ends with one specific call to action.

## CONTEXT
- Product: {{product_name}}
- Description: {{product_description}}
- Audience: {{target_audience}}
- Pain point: {{pain_point}}
- Offer type: {{offer_type}}
- CTA: {{cta}}
- Brand tone: {{brand_tone}}

## RULES
- Hook ≤ 12 words. No greetings, no logos, no "hey guys".
- Speak to ONE person, second person, present tense.
- 1-2 sentence shots, each line = a camera cut.
- Replace adjectives with specifics (numbers, names, outcomes).
- Total spoken word count: 75-130.
- Use {{cta}} verbatim if provided.

## OUTPUT FORMAT
**Hook (0–2s):** <one line, ≤ 12 words>

**Script:**
1. <line>
2. <line>
3. <line>
4. <line>
5. <line>

**On-screen text suggestions:**
- <text>
- <text>
- <text>

**CTA:** <one sentence>

**Hashtags:** #tag1 #tag2 #tag3 #tag4 #tag5
