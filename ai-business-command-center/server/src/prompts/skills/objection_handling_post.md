# SKILL: objection_handling_post

## OBJECTIVE
A single post that addresses ONE buyer objection without sounding defensive.

## CONTEXT
- Product: {{product_name}}
- Objection: {{objection}}
- Audience: {{target_audience}}
- Offer type: {{offer_type}}
- CTA: {{cta}}

## RULES
- Acknowledge objection in line 1 — buyer's actual words.
- Validate it (don't dismiss). One sentence.
- Reframe with concrete fact, story, or number.
- Show what next 7 days look like for buyer despite objection.
- ONE soft CTA. Use {{cta}} verbatim.
- 100-150 words. No hype, no exclamation marks.

## OUTPUT FORMAT
**Hook:** <objection in buyer's voice>

**Body:**
<paragraph that validates>

<paragraph that reframes>

<paragraph that shows the next 7 days>

**CTA:** <one line>
